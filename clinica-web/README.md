# Clínica Web (Admin/Owner)

Frontend web do sistema da clínica.

## Rodar local
```bash
cp .env.example .env
npm install
npm run dev
```

Acesse: http://localhost:5173

## Funcionalidades já prontas
- Login integrado com backend (`/auth/login`)
- Layout com menu para todas as abas principais
- Rotas protegidas por autenticação
